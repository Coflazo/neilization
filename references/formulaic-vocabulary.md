# Formulaic Vocabulary

This list is not a detector-bypass list. It is a cleanup list for words and phrases that often make prose vague, promotional, or over-processed. Replace them only when they weaken the sentence. If a term is technically correct or appears in a quoted source, keep it.

## Weak Verbs

Prefer direct verbs.

| Avoid when vague | Prefer |
|---|---|
| delve into | examine, explain, study, look at |
| leverage | use |
| utilize | use |
| facilitate | help, enable, make easier |
| optimize | improve, tune, make faster |
| underscore | show, stress, make clear |
| foster | build, support, encourage |
| bolster | strengthen, support |
| enhance | improve, sharpen |
| illuminate | show, reveal, clarify |
| navigate | handle, work through |
| unravel | explain, untangle |
| elucidate | explain |
| encompass | include, cover |
| discern | see, identify |
| transform | change, reshape |
| revolutionize | upend, change, replace |

## Inflated Adjectives

Cut unless the source proves them.

| Avoid when unsupported | Prefer |
|---|---|
| pivotal | turning-point, central, important |
| vital | necessary, essential |
| robust | solid, reliable |
| innovative | new, unusual |
| seamless | smooth |
| comprehensive | full, thorough |
| nuanced | complex, subtle |
| cutting-edge | latest, new |
| groundbreaking | first, new, major |
| transformative | major, changing |
| profound | deep, serious |
| multifaceted | complex |
| holistic | whole, full |
| inherent | built-in |
| invaluable | useful, essential |
| stark | clear, sharp |

## Abstract Nouns That Hide The Claim

| Avoid when empty | Prefer |
|---|---|
| landscape | field, system, market, scene |
| realm | area, world |
| tapestry | mix, pattern, network |
| transformation | change |
| testament | evidence, sign |
| synergy | overlap, combined effect |
| underpinnings | basis, roots |
| endeavor | work, project |
| insights | findings, lessons |
| implications | effects, consequences |
| perspectives | views, angles |
| interplay | relationship, tension |

## Decorative Transitions

Use the logical relationship instead.

| Avoid | Prefer |
|---|---|
| furthermore | also, and, more importantly |
| moreover | also, in addition |
| consequently | so, as a result |
| thus | so |
| nonetheless | but, still |
| additionally | also |
| subsequently | then |
| importantly | the important part is |
| notably | worth noting, or cut |
| essentially | basically, or cut |
| ultimately | in the end, or cut |

## Stock Openers And Closers

Usually cut the phrase and keep the claim.

- It is worth noting that
- It is important to note that
- In today's world
- In today's fast-paced world
- Based on the information provided
- First and foremost
- In conclusion
- In summary
- All in all
- To wrap up
- At the end of the day
- This serves as a reminder
- This highlights the importance of
- This reflects a broader trend
- This captures the essence of

## Public-Science Specific Warnings

Do not replace vague academic prose with empty wonder:

- awe-inspiring
- mind-blowing
- cosmic dance
- mysteries of the universe
- unlock the secrets
- journey through
- hidden wonders

Wonder must come from the mechanism, the scale, or the consequence.
